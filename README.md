

# Sudoku Solver

Sudoku Solver is a Python library to solve sudokus with an eas.


## Usage
### Uing the Gui
Mac:
```
python3 main.py
```

Windows:
```
python3.exe main.py
```
###Using the solver Package:
To solve just pass all arguments in the solving function:

```
loese_sudoku(gi)
```
**gi** is an array[9][9] witch holdes the unsolved sudoku.
